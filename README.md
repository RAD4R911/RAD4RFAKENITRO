# RAD4RFAKENITRO

RAD4RFAKENITRO is a BetterDiscord-style plugin package for Nitro-like client features, packaged under the RAD4RFAKENITRO name.

## Features

- Custom screen share quality, FPS, and bitrate controls
- Emoji and sticker bypass options
- Profile theme, banner, decoration, and effect tweaks
- Client theme and app icon options
- Clip upload bypass helpers
- Extra profile and context menu tools

## Install

1. Download `RAD4RFAKENITRO.plugin.js`.
2. Put it in your BetterDiscord plugins folder.
3. Restart Discord or reload plugins.
4. Enable `RAD4RFAKENITRO` from the Plugins page.

For Vencord users, this is a BetterDiscord plugin and needs a BD compatibility layer such as BetterVencord's BD Compatibility Layer.
